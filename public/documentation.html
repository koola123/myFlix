Documentation & Objective

To build a server-side component of a movie(s) application. The web application will provide users with access to information about different movies, directors, and genres.

Users will be able to sign up, update their personal information - and create a list of their favorite movies as well.

*** User Stories ***

  • As a user, I want to be able to receive information on movies, directors, and genres so that
    I can learn more about movies I've watched or am interested in.
  • As a user, I want to be able to create a profile so I can save data about my favorite movies.

*** Essential Features provided by the app ***

  • Returns a list of ALL movies to the user.
  • Returns data about a specific genre, director, image URL about a single movie by title to the user.
  • Returns data about a genre(description) by name/title (e.g., Thriller).
  • Returns data about a director like bio, birth year, death year by name.
  • Allows new users to register.
  • Allows users to update their user info(username, password, email, date of birth).
  • Allows users to add a movie to their list of favorites.
  • Allows users to remove a movie from their list of favorites.
  • Allows existing users to deregister.

Used Tech Stack:

MERN (MongoDB, Express, React, and Node.js)

URL endpoints:

Request: URL: HTTP METHOD: Request Body Data Format: Response Body Data Format:
Get a list of ALL movies /movies GET NONE JSON object holding data about all movies.
Get data about single movie /movies/[Description]/[Genre]/[Director]/[Image URL] GET NONE JSON object holding data about a single movie.
Get genre by name or title /movies/genre/[Description] GET NONE JSON object holding data about a movie genre.
Get data about a director /movies/director/[Bio]/[Birth Year]/[Death Year] GET NONE JSON object holding data about a movie director.
Register as a new user /users/[ID] POST --- TEXT MESSAGE that the user is registered now.
Update personal information /users/[ID]/[Username]/[Password]/[Email]/[Date of Birth] PUT --- TEXT MESSAGE that the personal user information was updated.
Add movies to favorites /movies/[Favorites] POST --- TEXT MESSAGE that the selected movie was added to the favorites list.
Remove movies from favorites /movies/[Favorites] DELETE --- TEXT MESSAGE that the specified movie has been deleted.
Deregister /users/[ID] DELETE --- TEXT MESSAGE that the registered user has been deleted.
